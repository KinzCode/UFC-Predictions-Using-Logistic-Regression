Predicting who will win and lose a fight in the UFC based on their hisotrical performances using logistic regression.
	
## Technologies
Project is created with:
* Python 3.7.8
  * Pandas 1.0.3
  * Numpy 1.19.4
  * SKLearn 0.24.1
